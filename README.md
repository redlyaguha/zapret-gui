# zapret-gui

Графическая оболочка для управления zapret на Windows. Приложение помогает установить zapret, выбрать стратегию обхода DPI, запускать её процессом или службой, управлять фильтрами, тестами, логами, автозапуском и обновлениями без ручного редактирования bat-файлов.

Готовые portable `.exe`-сборки публикуются в [GitHub Releases](https://github.com/redlyaguha/zapret-gui/releases).

## Возможности

- Первый запуск с выбором папки данных и установкой zapret или выбором существующей папки с `service.bat`.
- Вкладка DPI с выбором `.bat`-стратегии, режимом `Процесс` / `Служба`, статусом запуска и подробностями аргументов.
- Управление Game Filter, IPSet, обновлениями DPI, диагностикой, очисткой cache-папок Discord и встроенными тестами zapret.
- Встроенные тесты zapret через `utils/test zapret.ps1`: все конфиги или выбранные `.bat`.
- Системный трей с теми же состояниями, что в DPI: `Работает`, `Отключено`, `Запускается`, `Останавливается`, `Ошибка`.
- Одна tray-кнопка запуска/остановки: запускает последнюю успешно включённую стратегию с сохранённым режимом или отключает zapret.
- Автообновление zapret-gui через GitHub Releases с проверкой SHA256 для portable `.exe`.
- Автозапуск вместе с Windows: в окне или скрыто в трее.
- Опция всегда запускать zapret-gui от имени администратора через Windows Task Scheduler.
- Пользовательская папка данных `zapret-gui-config` для `config.json`, логов и временных файлов обновления.

## Требования

- Windows 10/11.
- Для portable `.exe` Python не нужен.
- Для запуска из исходников нужен Python 3.10+ и зависимости из `requirements.txt`.
- Для запуска стратегий как службы, встроенных тестов, работы с драйверами WinDivert и некоторых update-действий нужны права администратора.

## Установка

### Portable `.exe`

1. Откройте [Releases](https://github.com/redlyaguha/zapret-gui/releases).
2. Скачайте `zapret-gui-vX.Y.Z-windows-x64.zip`.
3. Распакуйте архив в удобную папку.
4. Запустите `zapret-gui.exe`.

При первом запуске приложение предложит выбрать родительскую папку для данных. По умолчанию используется `Documents\zapret-gui-config`.

### Из исходников

```powershell
git clone https://github.com/redlyaguha/zapret-gui.git
cd zapret-gui

python -m venv .venv
.\.venv\Scripts\activate

pip install -r requirements.txt
python main.py
```

## DPI

Вкладка DPI показывает найденные `.bat`-стратегии zapret. Можно выбрать режим запуска:

- `Процесс` - запускает выбранный `.bat` как обычный процесс.
- `Служба` - создаёт и запускает Windows-службу `zapret`.

После успешного запуска приложение сохраняет имя стратегии и режим запуска. Tray использует эти данные для кнопки “Запустить последнюю стратегию”.

В блоке “Инструменты” доступны:

- Game Filter: выключен, TCP+UDP, только TCP, только UDP.
- IPSet: загруженный список, отключить список, любой IP.
- Проверка обновлений DPI.
- Диагностика окружения.
- Очистка cache-папок Discord: `%APPDATA%\discord\Cache`, `Code Cache`, `GPUCache`.

Перед расширенными логами находится блок “Встроенные тесты zapret”. Он запускает `utils/test zapret.ps1`, позволяет выбрать тип теста и проверить все или отдельные `.bat`-конфиги.

## Tray

После закрытия окна приложение остаётся в системном трее, если это включено в настройках. Tray-меню показывает актуальный статус DPI и окрашивает его так же, как основная страница.

Единая кнопка tray работает по состоянию:

- если zapret работает - `Отключить`;
- если zapret отключён - `Запустить последнюю стратегию`;
- во время запуска/остановки кнопка временно недоступна.

## Настройки, папка данных и логи

Папка данных всегда называется `zapret-gui-config` внутри выбранной пользователем родительской папки. В ней хранятся:

- `config.json` - настройки приложения;
- `logs\dpi` - DPI/UI-логи: `zapret-gui-dpi-YYYY-MM-DD.log`;
- `logs\gui` - технические логи приложения: `zapret-gui-app-YYYY-MM-DD.log` и `app-update-helper.log`;
- `logs\telegram` - резерв под будущие логи Telegram-модуля;
- `updates` - временные файлы автообновления zapret-gui.

В настройках можно изменить папку данных, открыть её из “О приложении”, посмотреть размер логов и очистить все сохранённые логи.

## Автозапуск и права администратора

В “Поведение” можно включить запуск вместе с Windows и выбрать режим:

- `В окне` - обычное открытие главного окна;
- `В трее` - запуск скрытым в системном трее.

Если “Всегда запускать от имени администратора” выключено, используется обычная запись `HKCU\Software\Microsoft\Windows\CurrentVersion\Run`. Если включено, приложение создаёт задачу Windows Task Scheduler с run level `highest`.

При обычном запуске без прав администратора zapret-gui предлагает перезапуститься с повышением прав.

## Обновления zapret-gui

По умолчанию приложение проверяет stable-релизы zapret-gui не чаще одного раза в день. В настройках можно включить beta/prerelease и выполнить ручную проверку.

Автоустановка поддерживается только для portable `.exe`. Для установки релиз должен содержать:

- `zapret-gui-vX.Y.Z-windows-x64.zip`;
- `zapret-gui-vX.Y.Z-windows-x64.zip.sha256`.

При запуске из исходников zapret-gui не заменяет файлы автоматически и открывает страницу релиза.

## Приватность и сетевые запросы

zapret-gui хранит настройки и логи локально в выбранной папке данных и не добавляет телеметрию. Сетевые запросы выполняются для рабочих функций:

- GitHub Releases zapret-gui для проверки и загрузки обновлений;
- Flowseal/zapret-discord-youtube для установки или обновления zapret;
- IPSet и hosts-файлы Flowseal при соответствующих действиях пользователя.

Логи могут содержать пути к локальным папкам, имена стратегий, сообщения subprocess-команд и ошибки. Кнопка “Очистить логи” удаляет локальные log-файлы из папки данных.

## Сборка

На Windows portable `.exe` и zip-архив можно собрать командой:

```powershell
.\scripts\build_windows.ps1
```

Скрипт создаёт `dist\zapret-gui.exe`, архив `dist\zapret-gui-vX.Y.Z-windows-x64.zip` и SHA256-файл для автообновления.

Версия задаётся в `core/app_info.py` через `APP_VERSION`.
