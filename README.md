# zapret-gui

Графическая оболочка для управления zapret на Windows. Приложение помогает выбрать стратегию запуска, управлять фильтрами, обновлять списки и запускать диагностику без ручного редактирования bat-файлов.

> Проект находится в активной разработке. Готовые Windows `.exe`-сборки публикуются в GitHub Releases в виде portable zip-архива.

## Возможности

- Автоматическая установка zapret при первом запуске или выбор уже установленной папки.
- Просмотр доступных стратегий из `.bat`-файлов zapret.
- Запуск выбранной стратегии как обычного процесса или установка как Windows-службы.
- Остановка `winws.exe`, службы zapret и связанных драйверов перед обновлением.
- Отображение аргументов выбранной стратегии прямо в интерфейсе.
- Управление Game Filter, IPSet Filter и Auto-Update Check.
- Проверка и установка обновлений zapret из актуального релиза.
- Обновление IPSet-списка и hosts-файла.
- Запуск диагностики окружения.
- Запуск встроенных тестов zapret с выбором всех или отдельных конфигов.
- Очистка cache-папок Discord.
- Работа в системном трее со статусом zapret и быстрыми действиями включения/отключения.
- Хранение настроек и ежедневных логов в выбранной пользовательской папке.
- Автоматическая проверка обновлений zapret-gui через GitHub Releases с подтверждением установки.

## Требования

- Windows 10/11.
- Для готовой `.exe`-сборки Python не нужен.
- Для запуска из исходников нужен Python 3.10+ и зависимости из `requirements.txt`:
  - PySide6;
  - requests.
- zapret можно скачать через первый запуск приложения или указать уже существующую папку с `service.bat`.

Некоторые операции требуют прав администратора: запуск тестов, работа со службами, остановка/удаление драйверов WinDivert и установка zapret как службы.

## Установка готовой сборки

1. Откройте раздел [Releases](https://github.com/redlyaguha/zapret-gui/releases).
2. Скачайте архив `zapret-gui-vX.Y.Z-windows-x64.zip` из последнего релиза.
3. Распакуйте архив в удобную папку.
4. Запустите `zapret-gui.exe`.

При первом запуске приложение предложит выбрать родительскую папку для настроек и логов. По умолчанию используется `Documents\zapret-gui-config`.

Если вы планируете запускать тесты, управлять службами или устанавливать zapret как службу, запускайте `zapret-gui.exe` через `Run as administrator`.

## Установка из исходников

```powershell
git clone https://github.com/redlyaguha/zapret-gui.git
cd zapret-gui

python -m venv .venv
.\.venv\Scripts\activate

pip install -r requirements.txt
python main.py
```

При первом запуске приложение предложит:

1. скачать и установить zapret в выбранную родительскую папку;
2. или указать существующую папку zapret, внутри которой есть `service.bat`.

Выбранный путь zapret сохраняется в `config.json` внутри папки данных приложения.

## Как пользоваться

### Strategies

На вкладке `Strategies` отображаются найденные `.bat`-стратегии zapret. Можно выбрать режим запуска:

- `Run as process` - запустить выбранную стратегию как процесс;
- `Install as service` - установить и запустить стратегию как Windows-службу.

После выбора стратегии справа показываются аргументы запуска. Текущий статус подсвечивается в списке и дублируется в системном трее.

### Updates

Вкладка `Updates` проверяет актуальную версию zapret и умеет скачать обновление. Перед заменой файлов приложение пытается остановить zapret, `winws.exe` и службы/драйверы WinDivert, чтобы обновление не упёрлось в заблокированные файлы.

### Tools

Вкладка `Tools` содержит дополнительные действия:

- обновление IPSet-списка;
- обновление hosts-файла;
- диагностику;
- очистку cache-папок Discord;
- запуск тестов zapret.

## Системный трей

После закрытия окна приложение остаётся в системном трее. Через tray-меню можно показать/скрыть окно, включить последнюю успешно запущенную стратегию, отключить zapret или выйти из приложения. Tooltip и меню показывают текущий статус:

- `zapret-gui - работает`;
- `zapret-gui - отключено`.

## Настройки и логи

Файлы приложения хранятся в папке `zapret-gui-config` внутри выбранного пользователем каталога. В настройках можно перенести эту папку, посмотреть размер сохранённых логов и очистить файлы логов.

Логи пишутся ежедневно в `logs\zapret-gui-YYYY-MM-DD.log`.

## Обновления zapret-gui

Приложение проверяет обновления zapret-gui в фоне не чаще одного раза в день. Stable-релизы используются по умолчанию, beta/prerelease можно включить в настройках.

Автоустановка поддерживается только для portable `.exe`. При запуске из исходников приложение сообщает о новой версии и открывает страницу релиза.

Для автоустановки в GitHub Release должны быть опубликованы два asset:

- `zapret-gui-vX.Y.Z-windows-x64.zip`;
- `zapret-gui-vX.Y.Z-windows-x64.zip.sha256`.

## Структура проекта

```text
core/       логика управления zapret, службами, фильтрами и диагностикой
gui/        PySide6-интерфейс
updater/    загрузка и установка обновлений zapret
assets/     иконка приложения
scripts/    локальные скрипты сборки
main.py     точка входа
```

## Сборка `.exe` локально

На Windows можно собрать portable `.exe` и zip-архив командой:

```powershell
.\scripts\build_windows.ps1
```

Скрипт создаст или переиспользует `.venv`, установит зависимости и PyInstaller, соберет `dist\zapret-gui.exe`, затем упакует его в `dist\zapret-gui-vX.Y.Z-windows-x64.zip` и создаст рядом `.sha256` файл для автообновления.

Версия приложения задается в `core/app_info.py` через `APP_VERSION`.
